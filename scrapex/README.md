# 🦅 ScrapeX — AI Super Agent for Web & Social Media Scraping

[![Python](https://img.shields.io/badge/Python-3.10+-blue)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)
[![Open Source](https://img.shields.io/badge/Open%20Source-Yes-brightgreen)]()

**Open-source Firecrawl alternative.** Extract data from any website, social media platform, or API with AI-powered parsing. Built for developers who need reliable web scraping at scale.

## 🔥 Features

- 🌐 **Universal Web Scraping** — any website, any structure
- 📱 **Social Media Extraction** — Instagram, TikTok, Twitter/X, Facebook
- 🤖 **AI-Powered Parsing** — intelligent content extraction & structuring
- 📊 **Multiple Output Formats** — JSON, CSV, Markdown, HTML
- 🚀 **Async & Fast** — built on modern Python async patterns
- 🔐 **Proxy Support** — residential & datacenter proxies
- 📦 **Self-Hostable** — run on your own VPS or deploy to cloud

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/hemoyt/ScrapeX.git
cd ScrapeX

# Install
pip install -r requirements.txt

# Run
python scraper.py --url "https://example.com" --format json
```

## 🛠 Tech Stack

- **Language:** Python 3.10+
- **HTTP:** httpx, aiohttp
- **Parsing:** BeautifulSoup4, lxml, AI models
- **Browser Automation:** Playwright (optional)
- **Output:** JSON, CSV, Markdown
